# Just Another TicTacToe AI
An AI for the game TicTacToe using minimax tree with alpha-beta pruning.

# Usage
An executable .jar file has been provided. 

Alternatively, you can compile and run the source files yourself:
```bash
cd jattai
javac -d bin src/*
java -cp bin Main
```
